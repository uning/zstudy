小数点移动引起小数大小变化

小数点像个魔法棒——向右移变大，向左移变小！

小数点向右移动一位，小数就扩大到原来的 10 倍！

🤔 问题锚点

一只蚂蚁身长 0.005 米，把小数点向右移动三位变成 5 米——一只蚂蚁怎么可能身长 5 米？

原来，小数点移动会让数变大或变小！怎么变的？有没有规律？

A. 右移变大，左移变小
B. 左移变大，右移变小
C. 移动不改变大小
D. 只有向左移才变

🎯 本课学习目标

  1. 理解小数点向右移动一位、两位、三位……小数分别扩大到原来的 10倍、100倍、1000倍。
  2. 理解小数点向左移动一位、两位、三位……小数分别缩小到原来的 1/10、1/100、1/1000。
  3. 能运用小数点移动规律快速计算乘以10、100、1000 或除以10、100、1000。
  4. 初步感受位值原理——每个数字所在的位置决定了它的实际大小。
💡 规律不难，关键是多练习。记住：右移 = 乘，左移 = 除！

➡️ 小数点向右移

小数点向右移动 一位，小数就扩大到原来的 10 倍（即 ×10）。

理解：0.03 = 3个0.01，×10 = 30个0.01 = 3个0.1 = 0.3。每个计数单位都升了一级！

➡️ 右移两位、三位……

右移两位 = ×100

0.045

→ ×100 →

4.5

0.045 × 100 = 4.5

右移三位 = ×1000

0.045

→ ×1000 →

45

0.045 × 1000 = 45

规律：小数点向右移动 n 位 = 原数 × 10ⁿ
位数不够时在末尾补 0
例：3.6 × 1000 = ？小数点右移三位，位数不够→ 3.600 → 3600（末尾补两个0）

⬅️ 小数点向左移

小数点向左移动 一位，小数就缩小到原来的 1/10（即 ÷10）。

理解：45 = 45个1，÷10 = 4.5个1 + 5个0.1。每个数字的位值降了一级！

⬅️ 左移两位、三位……

左移两位 = ÷100

45

→ ÷100 →

0.45

45 ÷ 100 = 0.45

左移三位 = ÷1000

45

→ ÷1000 →

0.045

45 ÷ 1000 = 0.045

规律：小数点向左移动 n 位 = 原数 ÷ 10ⁿ
位数不够时在前面补 0
⚠️ 注意：向左移时，整数部分前面的"0"不能丢！比如 5 ÷ 100 = 0.05，不是 .05

🎉 课堂总结

  1. 小数点向右移动 1 / 2 / 3 位 → 扩大到原来的 10 / 100 / 1000 倍（×10/×100/×1000）。
  2. 小数点向左移动 1 / 2 / 3 位 → 缩小到原来的 1/10 / 1/100 / 1/1000（÷10/÷100/÷1000）。
  3. 位数不够时：右移末尾补 0，左移前面补 0。
  4. 这其实就是位值制的体现——每个数字在不同位置代表不同的大小。
