小数与单位换算

用小数点移动的方法，轻松玩转长度、质量、面积和货币单位换算

大化小 × 进率，小化大 ÷ 进率

🤔 问题锚点

小明的身高是 1.45 米，体检表上要填以厘米为单位——该填多少？

妈妈买了 0.5 千克苹果，标签上写的是500 克——这两个数有什么关系？

A. 1.45 米 = 14.5 厘米
B. 1.45 米 = 145 厘米
C. 0.5 千克 = 5 克
D. 0.5 千克 = 50 克
单位换算的本质就是小数点移动！

🎯 本课学习目标

  1. 掌握单位换算的基本方法：高级→低级 ×进率，低级→高级 ÷进率。
  2. 能熟练进行长度、质量、面积、人民币单位间的换算。
  3. 学会复名数（如 1米35厘米）与单名数（如 1.35米）之间的互化。
  4. 理解单位换算与小数点移动规律的内在联系。
💡 核心：看出进率，确定方向（×或÷），移动小数点——三步到位！

🔄 换算方法总览

高级 → 低级

× 进率

例：米 → 厘米

1.45 米 = 1.45 × 100 = 145 厘米

小数点右移两位

低级 → 高级

÷ 进率

例：厘米 → 米

145 厘米 = 145 ÷ 100 = 1.45 米

小数点左移两位

口诀：高化低 × 进率，低化高 ÷ 进率

📏 长度单位换算

千米(km) → 米(m) → 分米(dm) → 厘米(cm) → 毫米(mm)

每相邻两个单位进率是 10（千米↔米特殊：进率 1000）

3.6 千米 = ( ) 米

km→m：进率1000，×1000

3.6 × 1000 = 3600 米

85 厘米 = ( ) 米

cm→m：进率100，÷100

85 ÷ 100 = 0.85 米

2.05 米 = ( ) 厘米

m→cm：进率100，×100

2.05 × 100 = 205 厘米

30 毫米 = ( ) 厘米

mm→cm：进率10，÷10

30 ÷ 10 = 3 厘米

⚖️ 质量单位换算

吨(t) → 千克(kg) → 克(g)

1 吨 = 1000 千克，1 千克 = 1000 克

0.5 千克 = ( ) 克

kg→g：进率1000，×1000

0.5 × 1000 = 500 克

800 克 = ( ) 千克

g→kg：进率1000，÷1000

800 ÷ 1000 = 0.8 千克

2.35 吨 = ( ) 千克

t→kg：进率1000，×1000

2.35 × 1000 = 2350 千克

150 克 = ( ) 千克

g→kg：进率1000，÷1000

150 ÷ 1000 = 0.15 千克

💰 人民币单位换算

元 → 角 → 分

1 元 = 10 角 = 100 分

3 元 5 角 = ( ) 元

角→元：÷10

3 + 5÷10 = 3.5 元

12.8 元 = ( ) 元 ( ) 角

0.8元 × 10 = 8角

12 元 8 角

8 角 5 分 = ( ) 元

角→元÷10，分→元÷100

8÷10 + 5÷100 = 0.85

0.06 元 = ( ) 分

元→分：×100

0.06 × 100 = 6 分

📐 面积单位换算

平方千米(km²) → 公顷(ha) → 平方米(m²) → 平方分米(dm²) → 平方厘米(cm²)

1 km² = 100 ha，1 ha = 10000 m²，1 m² = 100 dm²，1 dm² = 100 cm²

⚠️ 面积进率是长度进率的平方！比如 1 m² = 100 dm²（不是 10）！

2.5 m² = ( ) dm²

2.5 × 100 = 250

360 cm² = ( ) dm²

360 ÷ 100 = 3.6

🔄 复名数与单名数互化

复名数（如 1米35厘米）⇄ 单名数（如 1.35米）

复名数 → 单名数

1 米 35 厘米

= 1 米 + 35 厘米

= 1 米 + 0.35 米

= 1.35 米

35÷100=0.35

单名数 → 复名数

2.6 千克

= 2 千克 + 0.6 千克

= 2 千克 + 600 克

= 2 千克 600 克

0.6×1000=600

💡 把复名数拆成两部分：大单位部分不变，小单位部分 ÷ 进率变成小数！

📊 常用进率速查表

类别 单位及进率 举例
长度km(1000)→m(10)→dm(10)→cm(10)→mm1 m = 100 cm
质量t(1000)→kg(1000)→g1 kg = 1000 g
人民币元(10)→角(10)→分1 元 = 100 分
面积km²(100)→ha(10000)→m²(100)→dm²(100)→cm²1 m² = 10000 cm²
时间时(60)→分(60)→秒1 时 = 60 分（不是十进制！）
⚠️ 注意：时间单位不是十进制，不能用小数点移动方法！需要单独记忆！

🎉 课堂总结

  1. 换算核心：高级→低级 ×进率，低级→高级 ÷进率。
  2. 本质是小数点移动：×10ⁿ = 右移n位，÷10ⁿ = 左移n位。
  3. 复名数互化：拆成整数部分+小数部分分别处理。
  4. 特别注意：面积进率是长度进率的平方；时间单位不是十进制。
  5. 三步法：看进率 → 定方向 → 移小数点。
