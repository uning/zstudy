📖 核心概念 · 引入

第二关：小数加法——小数点对齐（核心概念）

✅ 已经知道：整数加减法竖式要末位对齐（个位对个位，十位对十位）。

⚠️ 但问题是：小数有小数点，末位对齐会算错！比如 2.3 + 1.5 末位对齐 → 3+5=8 ✓，2+1=3 ✓，但换一个题 3.08 + 0.9 呢？

💡 所以要学：小数加法要对齐小数点，让相同数位对齐，然后再相加。

📝 竖式演示：2.3 + 1.5

❌ 错误（末位对齐）

  2.3
+ 1.5
──────
  3.8  ← 巧合对了！

但换个数就错！

✅ 正确（小数点对齐）

  2.3
+ 1.5
──────
  3.8  ← 个位对个位，十分位对十分位

小数点对齐 = 相同数位对齐！

🧪 小测试：3.08 + 0.9 = ?

先独立判断，再看诊断反馈。
📖 Module 1b

位数不同时怎么办？

比如 2.35 + 1.8，一个两位小数，一个一位小数，怎么对齐？

🔑 秘诀：位数不同时，在末尾补"0"凑齐！

  2.35    →    2.35
+ 1.8     →  + 1.80  ← 补一个0！
──────        ──────
                4.15
① 小数点对齐：把两个小数的小数点对齐写竖式
② 末尾补零：位数不够的末尾补0，让小数位数相同
③ 按位相加：从最低位（百分位→十分位→个位）逐位相加，满十进一
④ 点上小数点：结果的小数点要和上面两个小数点对齐

📖 Module 2

第三关：小数减法——退位不慌张

小数减法和加法一样要对齐小数点，但减法还需要退位，特别是整数减小数！

📝 例题：5 - 2.3 = ?

步骤①：整数补小数点          步骤②：退位借1
  5      →    5.0               5.0
- 2.3    →  - 2.3             - 2.3
──────       ──────            ──────
                                2.7

步骤③：十分位 0-3 不够 → 向个位借1 → 10-3=7
步骤④：个位 5 被借走1剩4 → 4-2=2
步骤⑤：点上小数点 → 2.7 ✅
① 整数补零：整数当被减数时，补上小数点和0（5 → 5.0）
② 小数点对齐：和加法一样，小数点对齐，相同数位对齐
③ 退位借1：哪位不够减，向前一位借1当10

📖 Module 3

第四关：小数加减混合运算

当遇到连加、连减或加减混合时，从左到右依次计算！

📝 例题：12.8 - 3.6 + 2.4 = ?

 第一步：12.8 - 3.6         第二步：9.2 + 2.4
   12.8                     9.2
  - 3.6                   + 2.4
 ──────                  ──────
    9.2                    11.6

 答案：12.8 - 3.6 + 2.4 = 11.6 ✅
⚠️ 运算顺序：从左到右！没有括号时，按顺序一步一步算
💡 每一步都要对齐小数点，不要看到混合运算就慌了
🔍 验算技巧：算完后可以用估算检查——12.8减3.6约等于9，再加2.4约等于11.4，和11.6接近，说明没错！

练习：4.5 + 3.27 - 2.8 = ?

从左到右，分两步算！

🏆 Summary

本课小结：小数加减法通关秘籍

回到开头的问题：3.5 + 2 = 5.5 ✓，不是3.7！ 因为要对齐小数点，把2看成2.0，3.5 + 2.0 = 5.5。

01 小数点对齐是核心！它等价于相同数位对齐，不能像整数那样末位对齐。
02 位数不同补零：如 2.35 + 1.8 → 1.8 补成 1.80，再按位相加。
03 退位借1当10：整数减小数先补零（5→5.0），不够减就向前借1。
04 混合运算从左到右：按顺序一步一步来，每步对齐小数点。
