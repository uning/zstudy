Concept

什么是角？

关键概念：从一点引出两条射线所组成的图形叫做角。这个点叫做顶点，两条射线叫做边。角通常用符号"∠"表示。
Classification

角的分类

记一记：锐角<90°=直角<钝角<180°=平角<周角=360°。1周角=2平角=4直角。
Tool

怎么量角？—— 量角器的使用

Core

垂线 —— 当两条直线相交成直角

生活实例：窗户的边框、地砖的接缝、十字路口、直角三角尺的两条直角边。
Core

平行线 —— 永不相交的两条直线

生活实例：铁轨的两条钢轨、斑马线的边缘、笔记本的横线、钢琴键盘的黑键排列。
Interactive

动手探索：画角、画垂线、画平行线

用 GeoGebra 工具：点击"直线"画线，"垂直"工具画垂线，"平行"工具画平行线，"角度"工具量角。

提示：点击左上角菜单可以画点、直线、垂线、平行线和量角度。
