空间向量与立体几何

空间直角坐标系、向量运算与立体几何证明

📐 数学高二·G11人教版自主学习
01 空间直角坐标系
在空间中任取一点O为原点，作三条两两垂直的数轴：x轴、y轴、z轴。空间中任意一点P对应唯一的有序数组(x,y,z)，称为点P的坐标。

两点距离公式：|AB| = \sqrt[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]
02 空间向量的概念与运算 概念
空间向量与平面向量类似，具有大小和方向。

加法：a\vec + b\vec = (x₁+x₂, y₁+y₂, z₁+z₂)
数乘：λa\vec = (λx, λy, λz)
数量积：a\vec·b\vec = x₁x₂ + y₁y₂ + z₁z₂ = |a\vec||b\vec|cosθ

向量垂直 \iff a\vec·b\vec = 0
03 空间向量的坐标表示
若A(x₁,y₁,z₁)，B(x₂,y₂,z₂)，则 AB\vec = (x₂-x₁, y₂-y₁, z₂-z₁)。

中点公式：M = ((x₁+x₂)/2, (y₁+y₂)/2, (z₁+z₂)/2)

向量的模：|a\vec| = \sqrt(x²+y²+z²)
04 向量法证明平行与垂直 证明
线面平行判定：平面的法向量n\vec与直线的方向向量v\vec满足 n\vec·v\vec = 0
线面垂直判定：直线的方向向量v\vec与平面的法向量n\vec满足 v\vec = λn\vec
面面平行：两个平面的法向量平行
面面垂直：两个平面的法向量垂直，即 n\vec₁·n\vec₂ = 0
05 异面直线与空间角
异面直线所成角：cosθ = |a\vec·b\vec| / (|a\vec||b\vec|)，θ∈[0,π/2]
线面角：sinθ = |v\vec·n\vec| / (|v\vec||n\vec|)，θ∈[0,π/2]
二面角：cosθ = |n\vec₁·n\vec₂| / (|n\vec₁||n\vec₂|)
