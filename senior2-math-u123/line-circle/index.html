直线和圆的方程

直线的方程、两直线位置关系与圆的方程

📐 数学高二·G11人教版自主学习
01 直线的倾斜角与斜率
倾斜角α∈[0,π)：直线向上的方向与x轴正方向所成的角。
斜率k = tanα (α≠π/2)
两点斜率公式：k = (y₂-y₁)/(x₂-x₁)

k>0时直线上升，k<0时直线下降，k=0时水平，k不存在时竖直。
02 直线的五种方程 公式
① 点斜式：y-y₀ = k(x-x₀)
② 斜截式：y = kx + b
③ 两点式：(y-y₁)/(y₂-y₁) = (x-x₁)/(x₂-x₁)
④ 截距式：x/a + y/b = 1
⑤ 一般式：Ax + By + C = 0 (A,B不同时为0)

理解各形式的使用场景和互化是关键！
03 两直线的位置关系 应用
平行：k₁ = k₂ 且 b₁ ≠ b₂（或一般式中A₁/A₂ = B₁/B₂ ≠ C₁/C₂）
重合：k₁ = k₂ 且 b₁ = b₂
垂直：k₁·k₂ = -1（一般式中A₁A₂ + B₁B₂ = 0）

距离公式：点P(x₀,y₀)到直线Ax+By+C=0的距离
d = |Ax₀+By₀+C|/\sqrt(A²+B²)
04 圆的方程 公式
标准方程：(x-a)² + (y-b)² = r²，圆心(a,b)，半径r。
一般方程：x²+y²+Dx+Ey+F = 0（D²+E²-4F>0）
圆心(-D/2, -E/2)，半径r = \sqrt(D²+E²-4F)/2
