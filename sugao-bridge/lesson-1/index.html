Concept 1

认识「正数」🟢

正数（> 0）是什么？

正数就是比 0 大的数，比如 +3、+8、0.5、100。

前面可以写 "+"（正号），但通常可以省略，所以 3 一般就写成 3，不写 +。

🌡️零上 20℃
记作 +20℃ 或 20℃
🏔️海拔 +8844 米
（比海平面高）
📈赚了 +50 元
（收入为正）
💡 记忆口诀：正数像"爬坡"，往大走；"+"号可省，数字本身就表示正！

Concept 2

认识「负数」🔵

负数（< 0）是什么？

负数就是比 0 小的数，比如 −5、−1.2、−100。

前面必须写 "−"（负号），不能省略！少写"−"就变成正数了。

❄️零下 10℃
记作 −10℃
🌊海拔 −155 米
（比海平面低）
📉亏了 −30 元
（支出为负）
💡 记忆口诀：负数像"下楼梯"，往小走；"−"号不离身，丢了就出错！

Concept 3

0 是分界线 ⚖️

0 既不是正数，也不是负数

它是正数和负数之间的分界点。

🟢0 的右边是正数
（比 0 大）
⚖️0 自己
中立不站队
🔵0 的左边是负数
（比 0 小）
💡 记忆口诀：0 是"裁判员"，正数、负数两边站，自己不算哪一队！

Concept 4

谁大谁小？看数轴！📏

比较大小的两条金规则

① 在数轴上，右边的数 > 左边的数。

② 正数 > 0 > 负数，所以任何正数都比负数大。

💡 例：−5 和 +3 比，+3 更大（−5 在左，+3 在右）。

Concept 5

相反意义的量 🔄

什么是一对"相反意义的量"？

生活中很多量是成对相反的：一个用正数表示，另一个就用负数表示，它们合起来就是相反意义的量。

正向（正数 +）负向（负数 −）
盈利 +200 元亏损 −200 元
向东走 +500 米向西走 −500 米
存入 +1000 元取出 −1000 元
零上 +8℃零下 −8℃
💡 记忆口诀：相反意义一对对，正号负号来配对；先定"正方向"，反着就加"−"。
