Concept 1

小数加法竖式 📐

核心法则：小数点对齐！

小数加法竖式中，小数点对齐就是相同数位对齐。

个位对个位、十分位对十分位、百分位对百分位……

对齐后，从低位算起，满十进一，最后在得数里对齐点小数点。

Concept 2

小数减法竖式 ➖

退位借1法

小数减法也是小数点对齐——同样遵循相同数位对齐。

哪一位不够减，就向前一位借1当10，加上本位再减。

💡 记忆口诀："小数减法看对齐，不够减了借个1。借1当10加上去，减去下位就可以！"

Concept 3

得数化简：末尾的0要去掉 ✂️

小数末尾的0

小数的末尾添上0或去掉0，小数的大小不变。

这是小数的基本性质。所以计算结果中，得数末尾的0要化简去掉。

2.30 → 2.3
去掉末尾的0
15.00 → 15
两个0都去掉
7.050 → 7.05
只去掉末尾的0
中间的0不能去！
💡 记忆口诀："小数末尾去掉零，大小不变记心中。中间零可不能动，化简只消尾随0！"

Concept 4

连加连减与简便运算 🧮

整数运算定律在小数中同样适用！

加法交换律：a + b = b + a

加法结合律：(a + b) + c = a + (b + c)

减法的性质：a − b − c = a − (b + c)

例如：3.6 + 2.8 + 6.4 可以先用交换律变成 (3.6 + 6.4) + 2.8 = 10 + 2.8 = 12.8

9.5 − 2.3 − 3.7 可以变成 9.5 − (2.3 + 3.7) = 9.5 − 6 = 3.5

💡 记忆口诀："连加可以把序调，连减括号变加号。整数规律到小数，简便运算更快了！"

Summary

课堂小结 📝

回顾一下，今天学到了什么？

1
小数点对齐：小数加减竖式的关键是小数点对齐——也就是相同数位对齐，低位算起，得数对齐点小数点。
2
退位借1：哪一位不够减，向前一位借1当10，加上本位再减。
3
得数化简：计算完别忘了去掉得数末尾的0（如 3.20 → 3.2，15.00 → 15）。中间和开头的0不能去掉！
4
运算定律：加法交换律、结合律、减法的性质在小数中同样适用。连减可以变加括号（a−b−c=a−(b+c)），能凑整的先算。
