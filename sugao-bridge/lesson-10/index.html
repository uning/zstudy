Concept 1

小数的意义 🔍

小数是分母为 10、100、1000… 的分数的另一种写法：

🔹 1/10 = 0.1（一位小数），计数单位 0.1

🔹 1/100 = 0.01（两位小数），计数单位 0.01

🔹 1/1000 = 0.001（三位小数），计数单位 0.001

🔹 例：0.7 = 7/10，0.25 = 25/100，0.138 = 138/1000

Concept 2

小数的读写 ✍️

读小数：先读整数部分（按整数），再读小数点（读作"点"），最后依次读出小数部分的每个数字。

🔹 3.14 读作：三点一四

🔹 0.05 读作：零点零五（中间的 0 要读出来）

🔹 10.08 读作：十点零八

📌 写小数：对应数位写数字，缺位用 0 占位。

Concept 3

小数的性质 ⚖️

小数的末尾添上 0 或去掉 0，小数的大小不变。

🔹 0.3 = 0.30 = 0.300

🔹 化简：0.70 = 0.7，10.0 = 10（整数末尾 0 一般去掉）

🔹 改写：0.4 = 0.40（凑两位小数），便于比较、计算

⚠️ 是末尾的 0，中间的 0 不能去（0.05 ≠ 0.5）。

Concept 4

比较大小与单位改写 🔁

① 比较大小：先比整数部分，再依次比十分位、百分位…

· 例：3.14 ＞ 3.08（整数相同，十分位1＞0）

② 单位改写（进率 10、100、1000）：

· 米↔厘米：×100 / ÷100，如 1.2 米 = 120 厘米

· 元↔分：×100，如 3.05 元 = 305 分

Summary

课堂小结 📝

1 意义：分母是10/100/1000的分数写成一位/两位/三位小数。
2 计数单位：0.1、0.01、0.001，对应十分位/百分位/千分位。
3 性质：末尾添0去0大小不变；化简与改写靠它。
4 比较改写：先整数后小数位；单位换算用进率。
